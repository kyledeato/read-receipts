## General Information
- Allows users to create a username and enter a chat room.
- When more messages come up the paper rises like a receipt.


## Technologies Used
- Spring Boot
- Web Sockets
- Java Server Pages
- Spring Tool Suite
- Apache Tomcat


## Features
- Ability to create a username
- Instant chat messaging
